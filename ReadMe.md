依赖库：

1）selenium

solution: python -m pip install selenium

2) requests

solution: python -m pip install requests

安装方法：

1) 将selenium目录拷贝到D盘根目录下

2）安装Python3.7并设置好相应的环境变量

3）见依赖库

4）安装firefox最新版

运行方法：

在WishSpider目录里打开cmd并运行sel.py（python sel.py)

运行完后，相关数据保存在transaction.csv中

打开Excel新建表格，选择数据，选择自文本，选择transaction.csv

1）选择下一步

2）取消tab键，选择逗号

3）选择下一步

4）选择完成

5）选择确定
